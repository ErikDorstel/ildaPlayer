# ildaPlayer
ESP32-S3 based ILDA Player
#### Features
* plays ILDA files from SD Card
* Web UI over WiFi
#### MCP4822
* GPIO 10 - CS
* GPIO 14 - LDAC
* GPIO 16 - SCK
* GPIO 17 - MOSI
* GPIO 18 - MISO
#### Colors
* GPIO 4 - Red
* GPIO 5 - Green
* GPIO 6 - Blue
* GPIO 7 - Shutter
#### SD Card
* GPIO 11 - MOSI
* GPIO 12 - SCK
* GPIO 13 - MISO
* GPIO 15 - CS
